# 朝野之争游戏规则
《朝野之争》是一款类狼人的**回合制**派对游戏。游戏的背景设定是古代朝廷。但与传统的杀人游戏以及狼人不同的是，除了**角色**以外，还有**派系**之分。

角色与派系好坏无关。类似三国杀，角色是公开的，但是派系是隐藏的，只有自己知道。玩家要做的就是找到与自己一派的玩家，并且共同实现派系目标。

---
## 角色
### 皇帝：
仅限始皇帝，后来登基的保留之前角色的属性  

技能【君要臣死，臣不得不死】【白昼】：  
- 在每日早朝可以随意裁决1人。  

技能【圣旨】【白昼】：  
- 每日早朝可以替换太子、将军、宰相、东厂大都督。只能替换其中一个。  
- 不能与【君要臣死，臣不得不死】同时执行。  

技能【皇家侍卫】【黑夜】【被动】：  
- 无法被刺杀


### 皇子：
技能【弹劾】【白昼】：
	弹劾一名非皇帝角色。

技能【政变】【白昼】：  
	可以发起政变。政变是篡位的存在。  

### 大内高手：
技能【保护】【黑夜】：
	保护一位其他玩家今晚不被刺杀。可以不发动。

技能【暗杀】【黑夜】:
	刺杀一名玩家。不可与【保护】同时发动。

技能【习武之人】【黑夜】【被动】：
	一个刺客干不掉

### 东厂大都督 Warden of East Factory
技能【东厂监督 Big Brother from East Factory】【黑夜】：
- 将角色抓入监狱。监狱里的角色不能行动。
- 监狱里的角色不受刺杀。
- 只能关押一个角色。
- 一个角色最多在监狱待2晚。

技能【东厂酷吏 Torture Master from East Factory】【黑夜】：
- 折磨并杀死被绑架的角色。不能立刻执行。
- 当一个角色被抓紧进狱时，只有第二回合晚上才能杀死该角色。

技能【戒备森严】【黑夜】：
- 不被刺杀

技能【放人】【白昼】
- 释放被关押的角色。皇帝可以要求你释放。


### 锦衣卫 Brocade Coat Guard（多个）
技能【监视】【黑夜】：
	监视一名玩家今晚的行动。可以得知该玩家今晚造访了谁/对谁发动了技能，以及谁对该玩家发动了技能。

技能【暗杀】【黑夜】:
	刺杀一名玩家。不可与【保护】同时发动。

技能【习武之人】【黑夜】【被动】：
	一个刺客干不掉

### 文官 Courtier（多个）
技能【朝野人心】【夜晚】：
- 调查一个玩家是否属于自己阵营。

技能【弹劾】【白昼】：
	弹劾一名非皇帝角色。

### 武将：
技能【习武之人】【黑夜】【被动】：
	一个刺客干不掉

技能【政变】【白昼】：  
	可以发起政变。

## 派系
#### 皇帝（正常）：
皇帝是孤家寡人，自成一派。皇帝的目标是存活到游戏结束，不管使用什么方式。虽然忠臣是皇帝的伙伴，但是也难免有昏君误杀忠臣。不过没关系，只要能活到最后，就是胜利！

#### 皇帝（暴君）：
与正常皇帝目标一致，要活到最后。唯一多了一个要求，就是杀掉所有其他角色！

#### 皇子（野心）：
具有野心的皇子希望有一天会登上帝位。不管是被皇帝名正言顺地立为太子，还是篡位。也就是说，要么被皇帝立为太子并活到最后，要么篡位（弑君）成为（正常）皇帝并且活到最后。

#### 皇子（忠）：
具有忠心的皇子不会篡位，但是也不拒绝成为皇帝。他只要活下来就行了。

### 忠臣：
忠臣们需要铲除所有对朝廷有威胁的派系。忠臣一开始不知道谁是队友。
如果（正常）皇帝死亡，忠臣失败。
如果皇帝是暴君，则需要扶持一位皇子上位！并且保证新皇帝活到最后。
只要（正常）皇帝活到最后，忠臣们就算胜利，不需要忠臣存活。

### 奸臣：
奸臣不在乎谁当权，只要情况对自己有利就好。笑（活）到最后才是王道！每一个奸臣都是各自为营，不能相信任何人。在与其他玩家胜利条件不冲突的情况下，奸臣不需要杀掉其他人。

### 反贼：
反贼是要颠覆朝廷的存在！推翻政权，杀掉所有皇族。反贼一开始知道谁是队友。

### 外戚：
外戚是皇子的母亲家族。
外戚们必须在游戏结束之前干掉现任皇帝，扶植其皇子登基，成为皇帝，并且让新皇帝活到最后。一旦自己的皇子成为皇帝，则目标和忠臣类似。

外戚知道队友。
