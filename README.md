## 角色
### 皇帝：
仅限始皇帝，后来登基的保留之前角色的属性  
智力：1  
武力：0  
技能 【君要臣死，臣不得不死】【白昼】：  
- 在每日早朝可以随意裁决与智力点数相同的人数。  
- 但是如果裁决超过一人，超过的人数会扣除智力点数。  

技能【圣旨】【白昼】：  
- 每日早朝可以替换太子、将军、宰相、东厂大都督。只能替换其中一个。  
- 智力点大于2时，可以与 【君要臣死，臣不得不死】同时执行。  
- 不然只能执行一个技能。  

技能【天子诏谕】【白昼】：  
	召见【不能上朝】的玩家上朝，倾听民间疾苦。无限人数。  

技能【皇家侍卫】【黑夜】：  
	派遣大内高手保护玩家。

## 阵营
#### 皇帝：
皇帝是孤家寡人。朝野的斗争只要顺应他的理念，都是可以发生的。皇帝的目标是存活到游戏结束，不管使用什么方式。

#### 皇子（野心）：
具有野心的皇子拥有一天会登上帝位。不管是被皇帝名正言顺地立为太子，还是篡位。如果皇子成为了太子，他的胜利目标自动转为皇帝。

#### 皇子（忠）：
具有忠心的皇子不会篡位，但是也不拒绝成为皇帝。他只要活下来就行了。

### 忠：
忠臣们需要铲除所有对朝廷有威胁的派系。忠臣不知道队友。不管谁是皇帝都可以。

### 奸（中立）：
奸臣不在乎谁当权，只要情况对自己有利就好。笑（活）到最后才是王道！每一个奸人都是各自为营，不能相信任何人。

### 反：
反贼是要颠覆朝廷的存在！推翻政权，杀掉所有皇族以及所有忠臣。反贼知道队友。

### 乱：
乱民们才不在乎江山社稷，杀光所有人才是目的。乱民不知道队友。



## 特殊派系：
### 外戚：
外戚是皇妃和其亲戚。外戚必须扶其皇子登基。外戚知道队友。

### 邪教：
邪教是来自东方的神秘力量，发源于民间却有登上朝堂的野心！邪教是可怕的传销组织，可以拉人入伙。邪教要么让皇帝宣布其为国教并坚持到对后，要么杀掉所有不皈依之异教徒。邪教头目可以给人传密信，说服其加入。


## 组织：
组织也派系不同，组织是公开的，每个玩家都知道谁属于哪个组织。

### 东厂：
头目：大都督
走狗：锦衣卫
大都督可以每晚指派锦衣卫去监视或者保护某人。
如果大都督死亡，皇帝指派一名太监成为下一任大都督。如果没有太监，东厂解散，锦衣卫被搁置，成为平民。
大都督自己也会武功，不会轻易被杀死（需要3个武功点）。
大都督的武功点：2
锦衣卫的武功点：2

### 兵权：
军队每晚可以保护一人。时机成熟可以发动政变。
头目：将军，武功 2
走狗：士兵，武功 1

### 大内高手：
保护或者杀人。
头目：皇帝，武功 0
走狗：大内高手，武功 2

### 皇子门客：
作为皇子的座上宾，自然是皇子的心腹。
皇子白天秘密招募任何一位地位比自己低的作为心腹。
心腹晚上可以给皇子密报自己都做了什么（皇子每晚最后行动，得到最多的信息）
太子可以有两位心腹。
